# OptiFlowz-Video-Platform

OptiFlowz Video Platform template with separate `Frontend` and `Backend` apps.

## Quick start

```bash
npm run install:all
npm run dev:backend
npm run dev:frontend
```

Frontend runs from [`Frontend`](/C:/Github%20Projects/OptiFlowz-Video-Platform/Frontend) and backend runs from [`Backend`](/C:/Github%20Projects/OptiFlowz-Video-Platform/Backend).
